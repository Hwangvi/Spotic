Entorno de Demostración Activo por Restricción de Cuota de Spotify

La API de Desarrollo de Spotify restringe el acceso exclusivamente a correos autorizados en su panel de control (Error 403 Forbidden). Para garantizar que puedas evaluar este portfolio sin bloqueos, el Backend (Spring Boot) ha activado un Modo de Contingencia inyectando datos simulados perfectamente estructurados. Sin mencionar la restriccion del uso de Spotify for developers para cuentas Premium.

Traza del Error interceptada en el Servidor:
org.springframework.web.client.HttpClientErrorException$Forbidden: 403 Forbidden on GET request for "https://api.spotify.com/v1/me": "Check settings on https://developer.spotify.com/dashboard, the user may not be registered."
